It is implementinon of CRUD operation using spring boot and Angular.

Prerequisites

eclipse IDE - Java EE
Java 17
Apache tomcat server
Postman to check API.
Node js
Angular CLI

To run this project downlaod the both folder.
For frontend use command - ng serve
For backend :
1. Import in eclipse IDE.
2. Then run as Java Application
Then hit the url on browser http://localhost:4200/employees

Note : Both server will be Start.
